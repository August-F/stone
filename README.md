# 🪨 岩石図鑑

火成岩・堆積岩・変成岩、計18種類の岩石データを探索できるStreamlitアプリです。

## スクリーンショット

| タブ表示 | サイドバーフィルター |
|---|---|
| 3カテゴリをタブで切り替え | 硬度スライダーと検索ボックス |

## 機能

- **3カテゴリ対応**: 火成岩 / 堆積岩 / 変成岩
- **カード表示**: 各岩石を名前・説明・硬度・色・用途付きで表示
- **硬度フィルター**: モース硬度のスライダーで絞り込み
- **検索**: 岩石名（日本語・英語）や説明文で検索
- **カラーテーマ**: カテゴリごとに色分け（火成岩=赤系、堆積岩=茶系、変成岩=青系）

## ファイル構成

```
.
├── app.py          # メインStreamlitアプリ
├── styles.py       # カスタムCSS・スタイル関数
├── data/
│   ├── __init__.py
│   └── rocks.py    # 岩石データ・検索関数
└── README.md
```

## セットアップ

```bash
pip install streamlit
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

## データ構造

`data/rocks.py` の各岩石は以下のフィールドを持ちます。

| フィールド | 型 | 内容 |
|---|---|---|
| `name` | str | 岩石名（日本語） |
| `name_en` | str | 英語名 |
| `description` | str | 説明文 |
| `hardness` | float | モース硬度 |
| `color` | str | 主な色 |
| `uses` | str | 主な用途 |

## 収録岩石一覧

**火成岩 (Igneous):** 花崗岩、玄武岩、黒曜石、軽石、安山岩、流紋岩

**堆積岩 (Sedimentary):** 砂岩、石灰岩、泥岩、礫岩、チャート、岩塩

**変成岩 (Metamorphic):** 大理石、片岩、片麻岩、石英岩、ホルンフェルス、蛇紋岩
